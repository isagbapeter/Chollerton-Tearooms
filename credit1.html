Security Considerations

At present, Chollerton tearooms has a lot of security risks. The first major concern relates to protecting its user's privacy and data. Currently, the Chollerton Tearooms website only requests basic information from its users. The website cannot enable users create password protected accounts, nor does it have a strong firewall that prevents hackers from accessing it’s server. If the website were to in future encourage more personalised experiences for its users, then all the above risks need to be considered. The website could make use of a good encryption software, such as Bitlocker and OpenSSL, that would help eradicate or minimize such risks. Another security risk for user’s is SQL Injection. This is a common form of cyberattacks on websites which enables the attacker to manipulate or extract date from the website’s database. Some prevention methods could be ensuring that all user input is thoroughly validated and sanitized, using parametrized queries or statements. The site currently has some basic validation, however, it's not secure enough therefore further validation must be added to prevent risks of data leak.

Lastly, the website is at a strong risk of suffering data loss due to hardware malfunction or hacking like earlier mentioned. This risk can be mitigated by ensuring regular backups of the website’s data onto secure backup software or hard drive.
